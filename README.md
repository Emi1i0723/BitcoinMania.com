# 🚀 BitcoinMania - Educación Crypto Moderna

BitcoinMania es una plataforma "All-in-One" diseñada para cerrar la brecha de información financiera en México, enfocándose en la educación sobre Bitcoin y el ecosistema de activos digitales.

## 🌟 Características Principales

- **📊 Monitor de Precios en Tiempo Real**: Seguimiento actualizado de Bitcoin (BTC), Ethereum (ETH) y Solana (SOL).
- **🧮 CryptoCALC**: 
  - Conversión precisa de criptomonedas a Pesos Mexicanos (MXN) usando una tasa de 17.79.
  - Comparador de valor entre diferentes activos digitales.
- **📚 BitcoinMania EDU**:
  - Artículos educativos curados con explicaciones simplificadas y analogías.
  - Sistema de **Flashcards Interactivas** para validación de conocimientos.
  - Soporte bilingüe (Español/Inglés) en todo el contenido.
  - **Accesibilidad LSM**: Videos integrados en Lengua de Señas Mexicana.
- **🧪 BitcoinLAB**: Acceso directo a un entorno virtual interactivo para simular inversiones.
- **📖 Sobre Nosotros**: Información detallada sobre nuestra misión nacida en el Hackathon Educativo de Bitcoin 2026.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **Componentes**: Shadcn/UI, Lucide Icons.
- **IA**: Google Genkit con modelos Gemini para la generación dinámica de contenido educativo.
- **Estado**: Context API para gestión multi-idioma.

## 🚀 Cómo empezar

1. Clona el repositorio.
2. Instala las dependencias: `npm install`.
3. Configura tus variables de entorno en `.env` (incluyendo tu `GOOGLE_GENAI_API_KEY`).
4. Ejecuta el servidor de desarrollo: `npm run dev`.
