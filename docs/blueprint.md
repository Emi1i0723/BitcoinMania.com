# **App Name**: CryptoMania

## Core Features:

- Secure User Authentication: Enable user registration and login via email/password using Firebase Authentication, supporting secure access to the platform.
- Language Toggle: Allow users to switch between Spanish and English for enhanced accessibility and user experience across the application.
- Cryptocurrency Price Monitor: Display static, constant prices for Bitcoin, Ethereum, and Solana in USD, with a user-interactive button to cycle through the different cryptocurrency displays.
- CryptoCALC - Currency Conversion: Provide a tool for converting cryptocurrency amounts to equivalent values in USD and Mexican Pesos (MXN) using fixed exchange rates.
- CryptoCALC - Crypto Comparison: Enable users to compare two cryptocurrencies, calculating how much of one cryptocurrency is equivalent to a given amount of another.
- AI-Powered Educational Content Generation: A generative AI tool that processes predefined educational and news content about cryptocurrencies, simplifying it into beginner-friendly explanations, analogies, and interactive flashcards for knowledge validation. This tool will use reasoning to decide when or if to incorporate specific educational content or current events into its output based on context or user interaction.
- Interactive Educational Sections: Organizes educational content into two main sections ('Todo lo que necesitas saber' and 'Novedades'). Each item displays content in a pop-up window, offering a language toggle (ES/EN), a 'like' button, and gamified flashcards for knowledge validation in the 'Todo lo que necesitas saber' section.
- Intuitive Dashboard Layout: A card-based dashboard design presenting key features (Price Monitor, CryptoCALC, CryptoMania EDU) in a clean, easily navigable dark-mode layout.

## Style Guidelines:

- Primary color: A vibrant yet deep digital purple (#9973FF), representing innovation and digital finance, providing good contrast in dark mode.
- Background color: A very dark, subtle bluish-grey (#17171A) for the default dark mode, promoting focus and a sleek aesthetic.
- Accent color: A bright, clear sky blue (#A0DBFF), analogous to the primary color, used for interactive elements and highlights to ensure visibility.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for a fintech platform.
- Use clean, minimalistic line icons that align with the modern fintech aesthetic, enhancing user comprehension without clutter.
- Dashboard content is presented within clearly defined, modern cards for a clean, organized, and easily digestible information flow.
- Incorporate subtle animations for state changes, card hovers, and data updates (e.g., in the price monitor), ensuring a fluid and engaging user experience without being distracting.
